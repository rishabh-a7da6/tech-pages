🕵️ Forward Proxy
Sits between
Client → Proxy → Internet
Hides the client's identity from websites.
Client IP hidden:✅ Yes
Server IP hidden:❌ No
Use cases:Anonymity, filtering
Examples:Squid, 3proxy
🛡️ Reverse Proxy
Sits between
Internet → Proxy → Server(s)
Hides the server's identity from clients.
Client IP hidden:❌ No
Server IP hidden:✅ Yes
Use cases:Load balance, cache, WAF
Examples:Nginx, Cloudflare
📖 The Middleman Analogy
🕵️ Forward Proxy = Your personal assistant. You whisper "Get me info from WebsiteX." The assistant walks there — WebsiteX sees the assistant, not you.

🛡️ Reverse Proxy = A receptionist. You walk into the building. The receptionist says "I'll handle your request" — you never see which employee actually did the work.
