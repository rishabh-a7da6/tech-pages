🛡️ Cryptographic Defense Mechanisms
Digital Certs (SSL) Think of this as a verified digital passport. If Alice correctly verifies it, she knows strictly that the server is the real Bank. This is what breaks MITM natively!
Certificate Authorities Your browser has a pre-installed root list of trusted entities that verify these passports. If the passport doesn't trace back to a CA, the browser throws a massive red warning screen.
HSTS Protocol Strict Transport Security completely blocks users from ever clicking the malicious "Continue anyway" button on a fake certificate page for sensitive domains.
