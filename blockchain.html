How Blockchain Works

A blockchain is a chain of blocks where each block contains data and a hash — a digital fingerprint. Every block's hash depends on the previous block's hash, creating an unbreakable chain. Edit any block below and watch the entire chain shatter.

🔗 How It Works
The Hash Fingerprint
Each block's hash is computed from: Block # + Data + Previous Hash + Nonce. Change any single character and the hash becomes completely different.
The Chain Rule
Each block stores the previous block's hash. If Block 2's "Prev Hash" doesn't match Block 1's actual hash, the chain is BROKEN.
